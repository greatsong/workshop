배경 · 왜 중요한 주제인가

데이터·AI는 특정 교과가 아니라 공통 소양

  • 2022 개정 교육과정은 디지털·AI 소양을 모든 학생의 기초 소양으로 명시한다.
  • 중학교 정보가 독립 필수가 되고, 초등 실과에 디지털·인공지능 영역이, 고교에 데이터 과학·인공지능 기초 등 과목이 신설됐다.
  • 데이터를 수집·시각화·해석하는 능력은 정보뿐 아니라 수학(자료와 가능성·통계)·사회(자료·인구·지리정보)가 공유하는 역량이다.
  • 공공데이터가 개방되어, 교사가 별도 준비 없이 실제 데이터로 수업을 설계할 수 있다.
  • 코딩이라는 진입 장벽이 생성형 AI로 낮아져, 교사가 직접 도구를 만들어 볼 수 있는 시점이 됐다.
연결 지도 · 실습 4단계와 교육과정

실습 한 번이 초·중·고 성취기준으로

워크샵 실습 4단계(배포·데이터·시각화·대시보드)는 각각 여러 교과·학교급의 성취기준과 맞닿는다. 학교급별 대표 성취기준만 추렸다. 실습 단계 또는 학교급을 눌러 연결을 확인하세요.

아래 표에서 행(실습 단계) 또는 열(학교급)을 누르면 해당 성취기준만 밝게 표시됩니다. 칩을 누르면 본문이 보입니다.
성취기준 칩을 누르면 전체 문장이 여기에 표시됩니다.
중요 이슈 · 데이터 윤리 — 실제 인구 데이터를 다루므로 개인정보와 공개 범위에 대한 판단이 함께 필요하다. 중학교 [9정04-05], 고등 [12소생03-04]가 이 지점에 직접 연결된다.

이후 · 어떻게 확장하는가

확장 경로

  • 소재 — 인구에서 급식·학교 설문·도서 대출·출결·지역 통계로 바꾼다.
  • 교과 — 사회(인구·지역), 수학(통계 단원), 국어(미디어·설문), 창의적 체험활동·동아리로 이식한다.
  • 수준 — 개인 저장(브라우저)에서 공유 데이터베이스(구글 시트), 실시간 협동 집계로 심화한다.
  • 학생 프로젝트 — 자유 주제 데이터 탐구·발표·공유, 학교생활기록부 연계.
  • 평가 — 통계적 탐구 과정과 산출물(앱) 공유를 수행평가로 삼는다.
진행하는 교사 관점 · 80분

어떻게 흘러가는가

  • 0–10분 · 오프닝
    참가자가 휴대폰으로 자기 학교를 지도에 표시. 즉시 참여하며 접속·배포 감각을 확인.
  • 10–20분 · 도입
    데이터로 답하는 질문 제시 → 인구 데모 → API 개념 설명.
  • 20–70분 · 실습(50분)
    인구 앱 4단계: 배포 → 데이터 → 시각화 → 대시보드. 단계마다 성공을 함께 확인.
  • 70–80분 · 마무리
    학교·수업 데이터로의 확장 안내(급식·설문 등), 질의응답.

진행 시 관리할 것

  • 사전 안내 — 깃허브 계정 가입·이메일 인증을 미리 마치게 한다.
  • 진도 편차 — 막히면 '완성 코드' 탭으로 따라오게 하고, 짝 도우미를 둔다.
  • 네트워크 — 그래프 라이브러리와 데이터를 인터넷에서 불러오므로 안정적인 와이파이가 필요하다.

점검 · 네 가지 포인트

좋은 · 와우 · 아쉬운 · 노력할 점

좋은 포인트

  • 결과물이 즉시 공개 URL로 남아 성취가 분명하다.
  • 가공된 예시가 아니라 행정안전부의 실제 인구 데이터를 쓴다.
  • 단계마다 'AI로 만들기 / 완성 코드' 두 경로로 낙오를 막는다.
  • 말로 지시해 코드를 바꾸는 과정을 직접 경험한다.

와우 포인트

  • 지역을 검색하면 우리 동네 인구 구조가 피라미드 그래프로 뜬다.
  • 문장 하나로 디자인·기능이 바뀐다.
  • 배포 후 자기 휴대폰으로 접속된다.

아쉬운 포인트

  • 데이터가 특정 시점(2026-05)의 스냅샷이라 '실시간 최신'은 아니다.
  • 그래프에 외부 라이브러리(Plotly)를 쓰므로 인터넷이 끊기면 표시되지 않는다.
  • 80분에 배포+실습을 다 넣으면 빠듯하고, 계정 미준비자가 많으면 지연된다.
  • 지역명 검색에서 동명·행정구역 변경 등으로 혼동이 생길 수 있다.

노력·관심할 점

  • 사전 안내(계정·브라우저·와이파이)로 도입 시간을 줄인다.
  • 워크샵 이후 실제 '학교 데이터'로 만들도록 후속 동기·자료를 준다.
  • 개인정보·데이터 윤리(수집·공개 범위)를 함께 다룬다.
  • 교과별 성취기준과 연결한 수업 설계 예시를 축적한다.
